<h1>忍者道</h1>

<h3>開発目的</h3>
昔からゲームやアニメなどの作品が好きで、自分のアイデアや世界観を形にしたいという思いがありました。特に、プレイヤー自身がコンテンツを創造し共有できるUGC（User Generated Content）の仕組みに魅力を感じ、ユーザーが自作ステージを投稿できる2Dアクションゲームの開発に着手しました。

<h3>開発期間</h3>
学部1,2年次（2023年10月〜2024年4月、約6ヶ月間）

<h3>開発環境</h3>
使用言語：C#
ゲームエンジン：Unity
ハードウェア：Windows
その他：ミニドット絵メーカー３（ビジュアルデザインに使用）

<h3>制作物の概要</h3>
大学在学中、複数人と授業課題で制作したゲームに着想を得て、Unityを用いて個人で企画・設計・プログラミング・ビジュアルデザインの全工程を担当し、2Dアクションゲームとして新たに開発しました。最大の特徴は、プレイヤーがオリジナルのステージを自由に作成し、他のユーザーと共有できる点です。プログラミングだけでなく、キャラクターのイラスト制作やUIデザイン、アニメーション、エフェクトの実装など、ビジュアル面や操作性にも工夫を凝らしました。
